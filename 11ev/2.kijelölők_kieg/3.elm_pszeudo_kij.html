Pszeudo-osztályok és pszeudo-elemek

A dokumentumoknak lehetnek olyan elemei vagy az elemek részei, amelyek formázást igényelnek, de az eddigi kijelölőkkel nem tudjuk őket elérni, de nem tartozik hozzájuk önálló elem, pl. az állapotuk alapján akarjuk elérni (rámutattunk, ki van jelölve, letiltott, bejelölt), vagy a struktúrában elfoglalt helyük alapján szeretnénk kiválasztani, vagy az objektumnak csak egy részét szeretnénk formázni.

Pszeudo-osztályok

A pszeudo-osztályokat használjuk az egyes elemek bizonyos állapotban történő eléréséhez.
Két nagy csoportba soroljuk ezeket: dinamikus és struktúra szerinti elérés alapján.
Szintaxisa: szelektor:pszeudo-osztály { meghatározásblokk }

Dinamikus pszeudo-osztályok: állapot szerinti elérés

  • :link, :visited, :active - a linkek állapotai szerinti elérés
    példa: a:active { color: ....; }
  • :hover - az elemre történő egérrel való rámutatás állapota
    példa: div:hover { font-size: 200%; }
  • :focus - az elem a fókuszba kerül (pl. egy input tagba visszük a kurzort)
    példa: input[type="number"]:focus { background-color: yellow; }
  • :target - az elért célpont formázása (pl. belső, oldalon belül linkelt elemekhez)
    példa: a[href^="#fejezet"]:target { font-weight: bolder; }
  • :enabled - engedélyezett állapotú elemekhez (pl. űrlapokhoz)
    példa: input[type="text"]:enabled { color: green; }
  • :disabled - tiltott állapotú elemekhez (pl. űrlapokhoz)
    példa: input[type="text"]:disabled { background-color: red; }
  • :checked - bejelölt állapotú elemekhez (pl. űrlapoknál a "radio" és a "checkbox" típusokhoz)
    példa: input[type="text"]:disabled { background-color: red; }

Struktúrális pszeudo-osztályok: a HTML-struktúrában elfoglalt hely alapján történő elérés

  • :root - a html elem formázásához (a body ehhez képest lehet kisebb!)
  • :empty - olyan html elemekhez, amelyeknek nincs tartalma (pl. <div> </div>)
    példa: div:empty { width: 100px; height: 100px; border: 1px dashed red; }
  • :nth-child(x) - a megadott tagok közül az x. testvér
    (x lehet pl. 3n+2 alakú is, használható az odd (=páratlan) és az even (=páros) kifejezés is)
    példa-1: felsorolás negyedik bejegyzése: li:nth-child(4) { ... }
    példa-2: a táblázat páros sorai: tr:nth-child(even) { ... }
  • :first-child - a megadott tagok közül az 1. testvér
    példa-1: táblázat első sora: tr:first-child { ... }
    példa-2: az összes bekezdés első dőlt gyermeke: p > i:first-child
  • :last-child - a megadott tagok közül az utolsó testvér
    példa-1: táblázat utolsó oszlopa: td:last-child { ... }
    példa-2: a body elem gyermekei közül az utolsó: body :last-child { ... }
  • :nth-last-child(x) - a megadott tagok közül hátulról számolva az x. testvér
    példa: táblázatnak hátulról a 3. oszlopa: td:nth-last-child(3) { ... }
  • :only-child - olyan elem, amelynek nincs testvére vagyis a szülőnek ő az egyetlen gyermeke
    példa: olyan bekezdés, amelynek nincs testvére: p:only-child { ... }
  • :nth-of-type(x) - egy szülő gyermekeinél a megadott típusú tagok közül az x. testvér
    (x lehet pl. 3n+2 alakú is, használható az odd (=páratlan) és az even (=páros) kifejezés is)
    példa: div elemek közül minden harmadik testvér (lehet más típusú testvére (pl. p) is, ): div:nth-of-type(3n) { ... }
  • :first-of-type - egy szülő gyermekinél a megadott típusú tagok közül az 1. testvér
    példa: a szülő gyermekei közül az első p elem: p:first-child { ... }
  • :last-of-type - egy szülő gyermekinél a megadott típusú tagok közül az utolsó testvér
    példa: a szülő gyermekei közül az utolsó h2 elem: h2:last-child { ... }
  • :nth-last-of-type(x) - egy szülő gyermekinél a megadott típusú tagok közül hátulról számolva az x. testvér
    példa: a szülő gyermekei közül hátulról a 2. kép: img:nth-last-of-type(2) { ... }
  • :only-of-type - egy szülő adott típusú gyermeke, amelynek nincs testvére
    példa: a body olyan leszármazott bekezdései, amelyeknek nincs testvére: body p:only-child { ... }

Pszeudo-elemek

A pszeudo-elemeket használjuk arra, hogy az egyes elemek részeit el tudjuk érni.
Szintaxisa: szelektor::pszeudo-elem { meghatározásblokk }

  • ::first-letter - az elem első betűje
    példa: a div elemek első betűje: div::first-letter { ... }
  • ::first-line - az elem első sora
    példa: a bekezdések első sora: p::first-line { ... }
  • ::selection - a felhasználó által kijelölt rész (tartalom) formázása
    példa: a bekezdésekben a felhasználó kijelöl egy szövegrészt, akkor inverz legyen (fekete alapon fehér színű):
    p::selection { background-color: black; color: white; }
  • ::before - tartalmat helyez el az adott elem elé
    példa: a h4 címsorok elé elhelyezi a rajz.gif képet: h4::before { content: url("rajz.gif"); }
  • ::after - tartalmat helyez el az adott elem mögé
    példa: a div elemek mögé elhelyezi a rajz.gif képet: div::after { content: url("rajz.gif"); }
  • ::marker - lista jelének (számozásának vagy felsorolásjelének) formázásához
    példa: ::marker { color: green; font-size: 150%; }
