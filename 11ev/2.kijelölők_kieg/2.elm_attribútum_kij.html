Kijelölés a jellemzők alapján

Amikor jellemző kijelölőt alkalmazunk, akkor az adott HTML-elem attribútumainak megléte, illetve az azokban meghatározott értékek alapján történik a kiválasztás, tehát nem az objektum, hanem az objektum egyes jellemzőit veszi figyelembe a böngésző.

További információk: w3schools CSS Attribute Selectors

Jellemzőérték alapú kiválasztás: [attribútum]

Az [attribútum] választóval azokat az elemeket választja ki a böngésző, amelyeknek létezik a szögletes zárójelek között megadott tulajdonsága.

Példa: kiválasztja azokat a képeket, amelyeknek megadtunk szélességi értéket:
img[width] { ... }

A jellemző pontos értékének megadása: [attribútum="érték"]

Az [attribútum="érték"] választóval azokat az elemeket választja ki a böngésző, amelyeknek létezik a szögletes zárójelek között megadott tulajdonsága és pontosan azzal az értékkel rendelkezik, amelyet megadtunk.

Példa: kiválasztja azokat az input mezőket, amelyek number típusúak:
input[type="number"] { ... }

Részleges jellemzőérték: [attribútum~="érték"]

Az [attribútum~="érték"] választóval azokat az elemeket választja ki a böngésző, amelyeknek létezik a szögletes zárójelek között megadott tulajdonsága és az értékében szerepel a megadott szó. Ennek megfelelően a megadott érték előtt és mögött is szóköznek kell állnia!

Példa: kiválasztja azokat az input mezőket, amelyek number típusúak:
a[title~="SZTE"] { ... }

Jellemzőérték eleje: [attribútum^="érték"]

Az [attribútum^="érték"] választóval azokat az elemeket választja ki a böngésző, amelyeknek létezik a szögletes zárójelek között megadott tulajdonsága és annak értéke elején szerepel a megadott karaktersorozat. A karaktersorozatot bármilyen más karakterek követhetik!

Példa: kiválasztja azokat a div mezőket, amelyeknek az azonosítója a "fej" kifejezéssel kezdődik:
div[id^="fej"] { ... }

Ha a megadott kifejezést a kötőjel kell hogy kövesse, akkor a ^ jel helyett a | jelet kell használnunk. Ha pl. vannak blokkjaink "top" és "top-" kezdetű osztályokkal is, akkor kizárólag azokat, amelyeknél a "top"-ot kötőjel követi, a div[class|="top"] kijelölővel tudjuk szelektálni.

Jellemzőérték vége: [attribútum$="érték"]

Az [attribútum$="érték"] választóval azokat az elemeket választja ki a böngésző, amelyeknek létezik a szögletes zárójelek között megadott tulajdonsága és annak értéke végén szerepel a megadott karaktersorozat. A karaktersorozatot bármilyen más karakterek előzhetik meg!

Példa: kiválasztja azokat a p mezőket, amelyeknek az osztálya a "teszt" kifejezésre végződik:
p[class$="teszt"] { ... }

Jellemzőérték tartalma (részlete): [attribútum*="érték"]

Az [attribútum*="érték"] választóval azokat az elemeket választja ki a böngésző, amelyeknek létezik a szögletes zárójelek között megadott tulajdonsága és annak értékében valahol szerepel a megadott érték. A megadott karaktersorozatot megelőzhetnek és követhetnek tetszőleges további karakterek!

Példa: kiválasztja azokat a linkeket, amelyeknek elérési útjában szerepel a "menu" karaktersorozat:
a[href*="menu"] { ... }
